# System do zgłaszania problemów z dostępnością cyfrową

Aplikacja webowa służąca do zgłaszania oraz zarządzania problemami z dostępnością cyfrową w oparciu o wytyczne WCAG. System umożliwia tworzenie zgłoszeń oraz ich późniejsze przeglądanie i aktualizację statusu przez administratora.

## Funkcjonalności
- dodawanie zgłoszeń problemów z dostępnością
- wybór kategorii problemu w oparciu o wytyczne WCAG 2.2
- przypisanie statusu zgłoszenia
- przeglądanie listy zgłoszeń
- edycja i usuwanie zgłoszeń
- filtrowanie i wyszukiwanie zgłoszeń
- logowanie dla administratora

## Technologie
- PHP 8.x
- Laravel (framework)
- SQLite (baza danych)
- Blade (szablony widoków)
- Bootstrap (interfejs użytkownika)
- Docker (środowisko uruchomieniowe)
